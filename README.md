# CKAN Demo (demo-ckan-org)

CKAN Demo is a organization open-data portal running **CKAN**, exposing a standardized catalog API over ~13 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/demo-ckan-org/refs/heads/main/apis.yml)
